# PCN Calculator

It's a calculator. You know, for math.

**Repository:** https://github.com/pacanekk/PCN-Calculator

## Installation

```bash
pip3 install customtkinter
./build_rpm.sh
sudo dnf install pcn-calculator-*.noarch.rpm
```

Run it from the menu or type `pcn-calculator`.

## Updates

```bash
./build_rpm.sh
sudo dnf upgrade pcn-calculator-*.noarch.rpm
```
